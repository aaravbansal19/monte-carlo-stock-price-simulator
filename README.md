Monte Carlo Stock Price Simulation

Description: A Python Monte Carlo simulation that models stock price movements using random daily returns, statistical reports, and data visualization through a histogram. 

Overview: This project simulates through thousands of possible stock price outcomes by generating random daily returns based on a normal distribution. This project analyzes the stock price results through statistical forms of measurement and visualizes the distribution of all of the simulation's ending prices through a histogram.

Features:
- 10,000 stock price simulation (adjustable in code)
- 252 trading day simulation period (adjustable in code)
- Random daily return generation
- Profit and loss probability calculations
- Average, median, highest and lowest price analysis
- Standard deviation calculation
- Histogram visualization

Technologies Used:
- Python
- Matplotlib
- Statistics Library
- Random Library

Output : The program first prints out the starting information which includes the starting price, number of trading days, and number of simulations. The program then generates the statistical report including the average, median, highest, lowest, standard deviation, and the amount of stocks above and below the starting price. Then, the program prints out a probability report, showing the percent chance of the stock making profit, loss, staying the same, gaining 10%, gaining 20%, losing 10%, and losing 20%. 

Purpose : This project focuses on probability, statistics, and financial modeling concepts used in quantitative finance.
